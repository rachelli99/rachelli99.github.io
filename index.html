Projects

Here are some of my projects I've done from my internships, clubs, classes, hackathons... or just from my desire to make something fun :)

Cipher Salad

I developed this project as a cybersecurity mini lesson to teach kids the history of secret messages and modern ciphers. Through this experience, I improved my skills in React (for the frontend layout), Bulma (for styling and design), and Firebase (for sharble ciphers).

Data Flow

A fast and interactive debug console, created with jquery and chartjs, that produces a diverse set of charts for large scale csv, json, and xml input (> 10k entries). This was used as a tool to analyze the offsets, id, variable properties, and size used for forwarding technology in compiler optimization.

Gee-nomics

This is a program used to store and edit genome / DNA sequences, and allows users to present the genome of a new organism and identify all stored genomes that have a high percentage of matching DNA. This utilizes efficient searching and matching algorithms written in C++ with object oriented programming, as well as data structures (such as a trie) to maintain a library of genomes.

Tour.AR

This project is an IOS App to improve visitors' experience when touring UCLA by using an augmented reality Bruin introducing the various places on campus. By working on this, I gained valuable experience working as a group, where we implemented page functionality in Swift, loaded area-based introduction text based on location detection. In addition, I designed user experience using Figma, and resolved size conflict for the app to be responsive on different devices.

SR Evaluation

This paper evaluated three different machine learning based, super-resolution techniques used for MRI images. These include: Enhanced Deep Residual Networks (EDSR), Rapid and Accurate Imaging Super-Resolution (RAISR), and Wide Activation Deep Residual Networks (WDSR). I used LaTeX to write the paper itself, which has mean absolute error, mean square error, and peak noise to signal ratio for evaluating the algorithms. I utilized various weight normalization and batch normalization factor values to compare the results.

Work Experience

Tencent | Researcher | 2020.05 - 2020.08

Worked on Remote Beats, a project to monitor heart rate remotely using photoplethysmogram (PPG) signals. This is done by measuring the RGB channel of in the area of forehead and cheeks after face recognition, acquiring the signals, and generating the heart rate measurement. Signal processing involves applying a Butterworth filter to detrend and using a Fourier transform to calculate the frequency. Technologies I used included Javascript and Bootstrap to improve user experience, and TensorFlow trained MediaPipe facemesh model to improve detection accuracy.

Motigo | Ebook Development Intern | 2019.08 - 2019.09

I utilized a combination of Python, HTML, and CSS to create these ebooks. Lastly, I designed the user interface, and analyzed the user experience of each product by researching targeted user group characteristics, analyzing user feedback, and highlighting the advantages of interactive ebooks.
